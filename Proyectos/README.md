# Gestor de Alumnos

Proyecto de consola en Python para gestionar alumnos (alta, baja, modificación y búsqueda).

## Tecnologías
- Python 3
- Programación Orientada a Objetos

## Funcionalidades
- Agregar alumno
- Eliminar alumno
- Modificar datos
- Buscar por DNI
- Mostrar lista completa

## Cómo ejecutarlo
1. Clonar el repositorio
2. Ejecutar `main.py` con Python
